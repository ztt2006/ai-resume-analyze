## Rules
- 使用 pdfplumber 解析多页PDF，完整拼接全文
- 自动清洗：多余换行、空格、特殊乱码字符
- 调用大模型必须强制限定输出 JSON 格式
- 必选提取：姓名、电话、邮箱、联系地址
- 加分提取：学历、工作年限、项目经历、期望薪资
- 岗位JD关键词提取 + 简历相似度分析
- 输出 0-100 匹配综合评分

## Workflow
1. 上传PDF → 解析文本 → 文本清洗
2. AI结构化信息抽取
3. 输入JD → AI匹配计算得分
4. 结果入库+缓存
