# 项目亮点说明

## 1. 招聘管理不止于“解析”
系统不仅能做简历解析和 JD 匹配，还补充了真实招聘后台常用的候选人管理能力：
- 候选人阶段管理
- 优先级管理
- 标签与备注
- 最近匹配决策快照

## 2. 工作台可直接用于答辩展示
首页已经升级为招聘工作台，展示：
- 核心统计卡
- 招聘流程概览
- Top 候选人榜单
- 待处理动作
- JD 模板管理

## 3. 批量对比更适合做招聘决策
批量对比不仅输出排序，还输出：
- Top 候选人
- 平均分
- 推荐人数
- 高风险人数
- 每位候选人的下一步动作

## 4. 工程实现贴近真实项目
- FastAPI 分层架构
- SQLAlchemy 2.0 ORM
- PostgreSQL + Redis
- 统一响应封装与异常处理
- React 19 + Vite + shadcn/ui + Tailwind
- 前后端均提供 README 与可运行说明

## 5. 数据增强设计
候选人管理字段没有强依赖额外迁移，而是复用现有 `structured_data` 做增量持久化，适合当前项目快速演示、迭代和部署。
