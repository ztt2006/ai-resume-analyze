# 前端说明

## 技术栈
- React 19
- Vite
- shadcn/ui
- Tailwind CSS
- Axios

## 页面能力
- 首页 / 工作台
  - 简历上传
  - JD 输入与匹配
  - JD 模板管理
  - 招聘流程概览
  - Top 候选人榜单
  - 待处理动作
- 历史记录页
  - 关键词筛选
  - 阶段筛选
  - 优先级筛选
  - 简历历史 / 匹配历史
- 批量对比页
  - 多候选人选择
  - JD 模板快速使用
  - 排名结果摘要
- 简历详情页
  - 结构化信息
  - 原文 / 清洗文本切换
  - 招聘管理信息编辑
  - 近期匹配记录

## 本地运行
1. 复制 `.env.example` 为 `.env`
2. 安装依赖：`npm install`
3. 启动开发环境：`npm run dev`
4. 打包：`npm run build`

## 环境变量
- `VITE_API_BASE_URL=http://127.0.0.1:8000/api`

## 说明
- 页面采用后台管理系统布局：左侧侧边栏，右侧内容区
- 保持与后端统一 REST 风格
- 使用统一请求封装和错误拦截
